Show Google Apps Script Setup Guide

1. Google Apps Script open karein aur "New Project" par click karein.

2. Neeche diye gaye code ko copy karke script editor me paste karein:

function doPost(e) {
  var data = JSON.parse(e.postData.contents);
  PropertiesService.getScriptProperties().setProperty('dh_db', JSON.stringify(data));
  return ContentService.createTextOutput(JSON.stringify({status: 'success'}))
    .setMimeType(ContentService.MimeType.JSON);
}
function doGet(e) {
  var db = PropertiesService.getScriptProperties().getProperty('dh_db') || '{}';
  return ContentService.createTextOutput(db)
    .setMimeType(ContentService.MimeType.JSON);
}

3. Top-right me "Deploy" -> "New Deployment" click karein.

4. Type select karein: **Web App**.

5. Configure karein: "Execute as: **Me**" aur "Who has access: **Anyone**".

6. Deployment link copy karke yahan settings me paste karke save kijiye!
